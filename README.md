# TNS Support Hub

Internal reference tool for The Next Street (TNS) agents handling student and parent inquiries about driver education in Connecticut and Massachusetts.

**Live site:** https://the-next-street.github.io/Support_Helper/

## What's in it

- **State guides** — CT and MA permit, license, and training requirements for teens (16–17) and adults (18+)
- **Service info** — bundle pricing, à la carte options, and locations
- **Eligibility & refund calculators** — quick lookups for agents on the phone
- **Policy reference** — cancellations, tardiness, lesson requirements
- **Agent training** — call scripts (including guidance for students with autism spectrum disorder), customer service guidelines, and escalation procedures
- **FAQ** — common student/parent questions

## How it works

The hub is a single static HTML file (`index.html`) hosted on GitHub Pages. There is no build step, no backend, and no database — everything lives in that one file.

## Updating content

1. Open `index.html` in the editor of your choice (or edit directly on GitHub).
2. Make your changes.
3. Commit to `main`. GitHub Pages auto-deploys within ~1 minute.
